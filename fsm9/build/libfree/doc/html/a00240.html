Detailed Description

Configures and executes packet error rate tests. WiCE(tm) only.

Field Documentation

◆ op

uint8_t freespace_PerRequest::op

0: sets the frequency set for fixed-frequency PER tests. 1: starts a PER test.

◆ payload

uint8_t freespace_PerRequest::payload[5]

op == 0: Sets fixed channels for the test. All 5 0xFF will clear the fixed frequency state. op == 1: Starts a PER test of duration ((payload[1] * 256 + payload[0]) * 256) WiCE(tm) frames.
