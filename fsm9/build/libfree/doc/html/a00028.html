◆ freespace_openDevice()

LIBFREESPACE_API int freespace_openDevice ( FreespaceDeviceId  id)

Open a Freespace device for use. A device must be in the opened state before most other commands in this API can be called. This internally allocates the resources needed to communicate with the device. freespace_closeDevice frees those resources.

Parameters
idThe FreespaceDeviceID of an attached device to open
Returns
FREESPACE_SUCCESS if no errors
